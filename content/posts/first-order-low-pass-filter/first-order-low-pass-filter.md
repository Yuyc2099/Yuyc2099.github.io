# 一阶低通滤波与指数平滑

## 1. 两种算法

本文比较两种常见的离散低通滤波算法：

1. 指数平滑滤波器；
2. 相邻输入平均的一阶低通滤波器。

本文只讨论算法本身，所有公式和代码均使用普通实数，不涉及 Q 格式或具体项目实现。

## 2. 指数平滑滤波器

指数平滑滤波器的公式为：

```text
y[n] = α × x[n] + (1 - α) × y[n-1]
```

公式中的符号表示：

```text
n        当前采样序号
x[n]     当前输入
y[n]     根据本次输入计算出的当前输出
y[n-1]   上一次采样得到的滤波输出
α        输出向当前输入靠近的比例
```

如果采样周期为 `Ts`，那么 `y[n-1]` 就是大约 `Ts` 秒前的输出。

也可以写成增量形式：

```text
y[n] = y[n-1] + α × (x[n] - y[n-1])
```

其中：

```text
0 < α ≤ 1
```

`α` 表示每次采样时，输出向当前输入靠近的比例。

- `α` 越小：滤波越强，响应越慢；
- `α` 越大：滤波越弱，响应越快；

## 3. 相邻输入平均的一阶低通滤波器

该算法的公式为：

```text
y[n] = y[n-1] + K × ((x[n] + x[n-1]) / 2 - y[n-1])
```

公式中的符号表示：

```text
n        当前采样序号
x[n]     当前输入
x[n-1]   上一次采样时的输入
y[n]     当前滤波输出
y[n-1]   上一次滤波输出
K        输出向相邻输入平均值靠近的比例
```

如果采样周期为 `Ts`，那么 `x[n-1]` 和 `y[n-1]` 都是大约 `Ts` 秒前保存的状态。

在这种一阶低通算法中，`K` 通常不是随意填写，而是根据采样频率和期望截止频率计算：

```text
Ts = 1 / Fs
ωc = 2π × Fc
K  = ωc × Ts
   = 2π × Fc / Fs
   = 2 × 3.1415926 × Fc / Fs
```

各参数含义：

```text
Ts   采样周期，单位为秒（s）
Fs   采样频率，单位为赫兹（Hz）
Fc   期望截止频率，单位为赫兹（Hz）
ωc   截止角频率，单位为弧度每秒（rad/s）
K    离散滤波系数，没有单位
```

它们之间的关系可以按下面的顺序理解：

```text
程序多久计算一次滤波器
→ 得到采样周期 Ts 和采样频率 Fs
→ 根据希望保留多快的信号选择 Fc
→ 计算 K
→ K 决定每次计算时输出向输入平均值靠近多少
```

`K = 2πFc/Fs` 是采样频率远高于截止频率时的常用近似。当 `Fc` 接近 `Fs` 时，实际截止频率会与期望值产生较大偏差，需要使用更精确的离散滤波器设计方法。

它可以拆成两步理解。

第一步，对当前输入和上一次输入求平均：

```text
x_avg[n] = (x[n] + x[n-1]) / 2
```

第二步，让输出向平均输入靠近：

```text
y[n] = y[n-1] + K × (x_avg[n] - y[n-1])
```

因此，这个算法可以理解为：

```text
两点移动平均 → 一阶递推平滑
```

为了获得稳定且不振荡的单调响应，通常选择：

```text
0 < K ≤ 1
```

通用 C 实现：

```c
float first_order_lpf(float x, float *x_prev, float y_prev, float k)
{
    float y = y_prev + k * ((x + *x_prev) * 0.5f - y_prev);
    *x_prev = x;
    return y;
}
```

调用时由外部保存上次输入和上次输出：

```c
float x_prev = 0.0f;
float y = 0.0f;
float k = 0.1f;

y = first_order_lpf(x, &x_prev, y, k);
```

## 4. 两种算法最核心的区别

指数平滑直接跟踪当前输入：

```text
x[n]
```

相邻输入平均算法跟踪的是：

```text
(x[n] + x[n-1]) / 2
```

所以后者在一阶递推之前，额外增加了一次两点平均。

这会产生两个主要结果：

1. 对采样点之间快速正负交替的噪声抑制更强；
2. 对输入突变的初始响应更慢，并增加约半个采样周期的延迟。

## 5. 阶跃响应对比

假设：

```text
初始输入   = 0
初始输出   = 0
输入突然变为 100
α = K = 0.2
```

### 5.1 指数平滑

第一次计算：

```text
y[0] = 0 + 0.2 × (100 - 0)     = 20
```

### 5.2 相邻输入平均

第一次计算：

```text
平均输入 = (100 + 0) / 2       = 50

y[0] = 0 + 0.2 × (50 - 0)     = 10
```

因此，在系数相同、状态相同的情况下：

```text
指数平滑第一次输出       = 20
相邻输入平均第一次输出   = 10
```

后者第一次只看到了输入阶跃的一半，所以起步更慢。

如果输入继续保持 100，两种滤波器最终都会收敛到 100。

## 6. 高频噪声抑制

以一组正负交替变化的输入为例：

```text
100, -98, 96, -100, 98, -96, ...
```

相邻输入平均结果为：

```text
(100  + -98)  / 2 =  1
(-98  +  96)  / 2 = -1
(96   + -100) / 2 = -2
(-100 +  98)  / 2 = -1
(98   + -96)  / 2 =  1
```

原始噪声幅值约为 `96～100`，相邻输入平均后的幅值约为 `1～2`。

假设初始状态为 0，并取 `α = K = 0.2`，两种滤波器的输出对比如下：

| 输入 | 指数平滑输出 | 相邻输入平均的一阶低通输出 |
| ---: | ---: | ---: |
| 100 | 20.0 | 10.0 |
| -98 | -3.6 | 8.2 |
| 96 | 16.3 | 6.4 |
| -100 | -6.9 | 4.7 |
| 98 | 14.0 | 3.6 |
| -96 | -8.0 | 3.0 |

指数平滑输出仍随输入正负摆动；相邻输入平均的一阶低通输出变化更平缓。

所以：

```text
相邻输入平均算法对接近采样频率一半的噪声抑制更强。
```

## 7. 延迟与响应速度

相邻输入平均算法使用了上一次输入，因此会引入额外延迟。

可以近似理解为：

```text
额外延迟 ≈ 0.5 × Ts
```

其中 `Ts` 是采样周期。

例如：

```text
采样周期 Ts = 1 ms
额外延迟约为 0.5 ms
```

对于温度、母线电压和缓慢变化的传感器，这点延迟通常可以接受；对于高速电流环、快速保护和高带宽控制，该延迟需要纳入稳定性分析。

## 8. 参数与截止频率

### 8.1 截止频率 Fc 的含义

`Fc` 描述滤波器允许信号变化得有多快。它不是一道硬边界，不代表低于 `Fc` 的信号完全不变、高于 `Fc` 的信号立即消失，而是滤波器开始明显衰减信号的参考位置。

对于标准一阶低通滤波器，频率为 `f` 的正弦信号通过后，幅值比例近似为：

```text
幅值比例 = 1 / sqrt(1 + (f / Fc)^2)
```

以 `Fc = 10 Hz` 为例：

| 输入频率 | 输出幅值约为输入的 | 直观含义 |
| ---: | ---: | --- |
| 1 Hz | 99.5% | 变化很慢，几乎完整保留 |
| 10 Hz | 70.7% | 到达截止频率，已经明显衰减 |
| 20 Hz | 44.7% | 只保留不到一半幅值 |
| 100 Hz | 10.0% | 高频成分受到强烈抑制 |

在截止频率处：

```text
输出幅值 ≈ 输入幅值 × 0.707
输出功率 ≈ 输入功率 × 0.5
```

这个位置称为 `-3 dB` 点。一阶低通超过截止频率后，频率每增加 10 倍，幅值大约再降低到原来的 1/10，也就是约 `-20 dB/十倍频`。

`Fc` 也可以从时间响应理解。标准一阶低通的时间常数为：

```text
τ = 1 / (2π × Fc)
```

当输入突然从 0 跳到 1 时：

```text
经过 1τ：输出约达到最终值的 63.2%
经过 3τ：输出约达到最终值的 95.0%
经过 5τ：输出约达到最终值的 99.3%
```

因此：

```text
Fc 越低 → 时间常数越大 → 更平滑，但响应更慢
Fc 越高 → 时间常数越小 → 响应更快，但保留更多高频噪声
```

简单示例：假设 `Fs = 1000 Hz`、`Fc = 10 Hz`，表示每 `1 ms` 采样一次，滤波器从约 `10 Hz` 附近开始明显衰减。

```text
τ = 1 / (2π × 10) ≈ 15.9 ms
α = 1 - exp(-2π × 10 / 1000) ≈ 0.0609
K = 2π × 10 / 1000 ≈ 0.0628
```

输入发生阶跃后，输出约经过 `15.9 ms` 达到最终值的 `63.2%`。两个滤波系数都约为 `0.06`，表示每次采样时输出向目标值靠近约 `6%`。

以上是标准一阶低通的参考特性。离散滤波器会受到采样频率和离散化方法影响；当 `Fs` 远大于 `Fc` 时，实际结果通常更接近上述关系。

### 8.2 指数平滑系数

如果希望根据截止频率 `Fc` 和采样频率 `Fs` 计算指数平滑系数，可以使用匹配连续时间极点的方法：

```text
Ts = 1 / Fs
α  = 1 - exp(-2π × Fc × Ts)
```

也就是：

```text
α = 1 - exp(-2π × Fc / Fs)
```

当 `Fc` 远小于 `Fs` 时，可以近似为：

```text
α ≈ 2π × Fc / Fs
```

通用计算函数：

```c
#include <math.h>

float calculate_exponential_alpha(float cutoff_frequency, float sample_frequency)
{
    return 1.0f - expf(-2.0f * 3.14159265359f * cutoff_frequency / sample_frequency);
}
```

`expf(x)` 是 `<math.h>` 提供的单精度指数函数，用于计算 `e` 的 `x` 次方；函数名末尾的 `f` 表示参数和返回值均为 `float`。

## 9. 离散传递函数

本节用于进一步理解两种算法的频率特性，不影响前面内容的使用。

### 9.1 指数平滑

指数平滑的传递函数为：

```text
                 α
H(z) = ---------------------
       1 - (1 - α) × z^-1
```

它包含一个极点：

```text
z = 1 - α
```

### 9.2 相邻输入平均的一阶低通

其传递函数为：

```text
             K × (1 + z^-1)
H(z) = --------------------------------
        2 × [1 - (1 - K) × z^-1]
```

它包含：

- 一个极点：`z = 1 - K`；
- 一个零点：`z = -1`。

`z = -1` 对应离散系统的最高频率，也就是正负交替的采样序列。这解释了为什么相邻输入平均算法能完全抑制该频率分量。

当 `α = K` 时，两种算法的极点相同，但相邻输入平均算法多了一个位于 `z = -1` 的零点。

## 10. 状态与复位

指数平滑只需要保存：

```text
上一次输出
```

相邻输入平均算法需要保存：

```text
上一次输入
上一次输出
```

初始化或复位时，可以将历史输入和输出设为第一次输入，避免输出从 0 缓慢爬升。

复位相邻输入平均算法时，这两个状态应同时更新：

```c
void averaged_input_lpf_reset(float value, float *x_prev, float *y_prev)
{
    *x_prev = value;
    *y_prev = value;
}
```

如果只复位输出、不复位上一次输入，下一次计算仍会受到复位前输入的影响。

## 11. 优缺点对比

| 对比项 | 指数平滑 | 相邻输入平均的一阶低通 |
| --- | --- | --- |
| 基本输入 | 当前输入 | 当前与上次输入的平均值 |
| 保存状态 | 上次输出 | 上次输入、上次输出 |
| 突变响应 | 较快 | 初始响应较慢 |
| 高频抑制 | 一般 | 更强 |
| 最高离散频率 | 只能衰减 | 理论上完全抑制 |
| 额外延迟 | 较小 | 约半个采样周期 |
| 实现复杂度 | 更低 | 略高 |
| 参数 | `α` | `K`，也可由 `Fc/Fs` 近似计算 |
| 初始化和复位 | 简单 | 必须同时管理输入和输出状态 |
| 适合信号 | 转速、温度、负载等慢变量 | ADC、电压等高频噪声较明显的信号 |
